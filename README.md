(À utiliser dans un projet, comme master)
Dans l'ordre : sma3, puis cli, puis vid2, puis AssocSonVid.

sma3 crée les fichiers JSON.
cli crée des images à partir des fichiers JSON.
vid2 crée une vidéo à partir des images crées précédemment.
AssocSonVid crée une nouvelle vidéo ajoutant à la vidéo précédemment crée le son associé.
